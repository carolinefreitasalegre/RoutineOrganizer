# Routine Organizer

Aplicação em desenvolvimento para **organização e gerenciamento de rotinas** pessoais e profissionais.  
O projeto tem como objetivo facilitar o controle de tarefas, compromissos e prioridades, oferecendo uma interface simples e eficiente.

> ⚠️ **Status:** Projeto em construção.  
> Algumas funcionalidades e módulos ainda estão em fase de desenvolvimento e testes.

---

## Tecnologias Utilizadas

- **.NET 8 / ASP.NET Core**
- **Blazor WebAssembly**
- **Entity Framework Core**
- **SQL Server**
- **Radzen Components**
- **Azure DevOps** (gestão de tarefas e sprints)
- **GitHub** (espelhamento e versionamento)

---

## Estrutura do Projeto

- **RoutineOrganizer.Api** – API principal, responsável pela comunicação com o banco de dados e regras de negócio.  
- **RoutineOrganizer.Client** – Aplicação Blazor WebAssembly (frontend).  
- **RoutineOrganizer.Shared** – Modelos e DTOs compartilhados entre cliente e servidor.

---

## Funcionalidades Planejadas

- Cadastro e edição de tarefas e rotinas
- Definição de prioridade e status
- Controle de tempo e produtividade
- Visualização de progresso diário/semanal

## Gestão e Versionamento

Azure DevOps: controle de tarefas, backlog e sprints

GitHub: repositório público para código-fonte e histórico de commits
